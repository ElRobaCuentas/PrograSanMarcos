

2. La empresa Pris S.A. vende artículos electrodomésticos cuyos datos son código,
descripción, color, precio unitario. Los datos de los artículos deben guardarse en
un arreglo de estructuras de tamaño 100. Crear pseudocódigo necesario para:
a. Ingresar datos de un artículo a la vez al arreglo.
b. Eliminar un artículo del arreglo, el usuario indicará el código del artículo
correspondiente.
c. Mostrar los datos de un artículo del que se especifique su código.
d. Mostrar todos los artículos de precio unitario superior a S/.1000

Accion eliminarCodigo(Articulos artic[], Entero &c)
DV
    Entero i, cod, pos
Inicio
    Leer(cod)
    pos <- 0
    Para i desde 1 hasta c Hacer
        Si artic[i].codigo = cod Entonces
            pos <- i
        FSi
    FPara
    Si pos != 0 Entonces
        Para i desde pos hasta c - 1 Hacer
            artic[i] <- artic[i + 1]
        FPara
        c <- c - 1
    FSi
Fin

Accion mostrarArtCodigo(Articulos artic[], Entero c)
DV
  Entero i, cod
Inicio 
  Leer(cod)
  Para i desde 1 hasta c Hacer
    Si artic[i].codigo = cod Entonces
        Escribir(artic[i].codigo)
        Escribir(artic[i].descripcion)
        Escribir(artic[i].color)
        Escribir(artic[i].precioUnit)
    FSi
  FPara
Fin


d. Mostrar todos los artículos de precio unitario superior a S/.1000


Accion mostrarArticPreUnit(Articulos artic[], Entero c)
DV
  Entero i 
Inicio
  Para i desde 1 hasta c Hacer 
    Si artic[i].precioUnit > 1000 Entonces
        Escribir(artic[i].precioUnit)
    FSi

Fin
